HOW TO USE THIS DOCUMENT 

This document provides a list of all of the available filing codes that will appear in the filing code 
drop-down box. The codes are listed by case category and the type of filing (Case 
Initiation/Subsequent). 

The codes listed under Case Initiation for a case category will appear when selecting NEW 
CASE in File and Serve. 

The codes listed under Subsequent are those codes that will appear when filing into an existing 
case in that case category. 

The following cases types do not have a Case Initiation list because they must be 
conventionally filed: 

Offense Felony, Offense Misdemeanor, Offense Violation, Contempt (Punitive and Remedial), 
Procedural Matters – Extradition, Juvenile Delinquency (All), Misdemeanor Appeal from a 
Justice or Municipal Court
