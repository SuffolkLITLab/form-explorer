RULE 508.3.  DEFAULT JUDGMENT (Redline Version) 

Generally.  If the defendant does not file an answer to a claim by the answer
date or otherwise appear in the case, the judge must promptly render a default
judgment  upon  the plaintiff’s  proof  of  the  amount  of  damages.  A  default
judgment must comply with Rule 505.1. When a default judgment is signed,

property garnished exceeds the amount necessary to secure the debt, interest for one
