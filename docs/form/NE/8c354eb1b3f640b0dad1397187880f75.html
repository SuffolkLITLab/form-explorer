INSTRUCTIONS FOR COMPLETING THE DECREE(S) FOR NAME CHANGE OF A MINOR CHILD 

If you are changing more than one child’s name you need to complete a Decree for each child 
for whom you are requesting a name change. In other words, if you are changing three 
children’s names you will need to prepare three Decrees. The heading of each Decree should 
be the same as the Petition you filed containing all children’s names for whom you are 
requesting name changes. The body of each Decree should contain only the name of one 
child and you will need a Decree for each child for whom you are requesting a name change.
