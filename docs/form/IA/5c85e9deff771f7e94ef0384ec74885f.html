Rule 17.10—Form 14: Protected Information Disclosure 

You are responsible for ensuring that protected information is not included on documents or exhibits you file with the 
court.    Protected information includes personal or confidential information that should not be available to the public. 
The clerk of court will not review filings to determine whether you have made required omissions or redactions.   

Use this form to identify the full version of any protected information redacted in other documents you have filed. 
If you do not understand how to use this form, or if you should use this form, talk to an attorney.
