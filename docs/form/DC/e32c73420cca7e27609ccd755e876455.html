Instructions: Using this or a substantially similar form, the filer must provide a numbered description of each 
charge or credit set forth in the proposed accounting and identify the corresponding exhibit. The exhibits must be 
submitted, in order, with the itemized sheet. The exhibits must include documentation that clearly shows support 
for the corresponding charge or credit asserted. If it is not clear from the face of the documentation how it 
supports the line item credit or charge, then the exhibit should include a written explanation. The filer may use as 
many copies of this form as necessary. Total amounts of charges and credits must also be carried forward to the
